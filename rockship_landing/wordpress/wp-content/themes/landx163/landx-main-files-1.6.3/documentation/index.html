Overview

We represent our latest LandX Multipurpose Bootstrap 3 Landing Page wordpress theme. This is mainly a one page template theme but you can use as like other simple theme with blog and others page as well. You can add diffrent pages content in one page in easy way by admin panel. There are some special features in our theme like advanced theme options, unlimited icons, unlimited shortcodes, one page template and default template, full width page etc . Fully compatible >IE9+, Firefox, Chrome, Safari, Opera etc.  There is a rich documentation where you can find all of your questions answer.This documentation aims to give beginners and advanced users an overview of what is possible with the LandX WordPress theme. The LandX theme is built on bootstrap responsive format that automatically adapts to fit any screen devices : computers, laptops, tablets or smartphones. We hope that LandX will be your best selection.

Installing WordPress

WordPress is well-known for its ease of installation. Under most circumstances installing WordPress is a very simple process and takes less than five minutes to complete. Many web hosts now offer tools to automatically install WordPress for you. However, if you wish to install WordPress yourself just read rest of content this sections and finish your wordpress installation.

Wordpress Installation

Installation Theme

Once you download LandX Theme, unzip the main item. You will see the following folders:

1. landx.zip

2. Documentation

3. DummyContent.zip

LandX can be installed in two different ways:

1. Installing through wordpress.

2. Uploading through FTP (File transfer protocol).

Installation through Wordpress:

Installing a theme through wordpress is quite a simple process. After you have logged into wordpress account and are ready to go, in the admin panel’s sidebar navigate to appearance and then select themes. After that, click on the Add New tab and select upload from the navigation bar right below it. Click “choose file” button and locate your theme zip file (landx.zip) then hit install now. After LandX has been installed locate and click activate.

Screenshot can be heplfull for you:

 

Uploading through FTP (File transfer protocol):

1. Download the Theme archive and extract the main theme zip file it contains (landx.zip).

2. You may need to preserve the directory structure in the archive when extracting these files. Some softwares like "winrar" create a sub directory with the same directory name they have unzipped. uploading theme to your server via FTP with this structure is WRONG and you will face with some errors.

3. Use an FTP client (for example filezilla) to access your host web server

4. Transfer your unzipped theme to your server in the wp-content/themes

5. Once the theme is uploaded, you need to activate it. Go to Appearance > Themes and activate your uploaded theme.

 

Install Required and Recommanded Plugins:

After activation your theme you see a submenu in your admin Appearance > Install Plugins. Click to checkbox and select to install then install all plugins and active plugins.

 

Importing Demo Content

Below explanation is an step by step explanation on how to import our demo website content, please read carefully and if you face any problems that is not covered here, open a thread in our support forum.

1. Login into your WordPress Admin section (http://www.your-web-site.com/wp-admin)

2. Go to Tools -> Import in the blog’s admin panels.

3. Click on the WordPress and follow the instructions if Import plugin isn’t installed yet (3.0+).

4. Browse the included demoContent.xml (XML file) from main Zip Package > DummyContent.zip and Upload the XML file.

5. WordPress will then import each of the posts, page, custom post types, comments, and categories contained in this file into your site.

6. After this, you should also update the Menus by going to Appearance > Menus. Locate the Theme Locations box and select from the drop-down “Main Navigation” as respect to the locations the selectbox label indicates. Click the Save button under it to save the changes.

7. Done! You should now have the dummy/demo content similar from the Live Preview of the theme.

Building the menu

Wordpress menu from Appearance > Menus is only applicable for others multi pages like blog page, single page, search or category page. One page menu is added by diffrently. In one page template settings section you can find how to set menu for one page.

You can add your menu easily by landx. Here are the steps

  1. Go to Appearance > Menus
  2. Enter a name for your menu
  3. Click on Create Menu
  4. Now in the Theme Location select the menu you just created
  5. Click on Save
  6. Now tick the pages you want to add to menu and click Add to Menu
  7. When you added all pages to your menu, you can drag/drop and order pages.
  8. At last click Save Menu, thats it.
